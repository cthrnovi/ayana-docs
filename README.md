# ayana-docs

Shareable HTML documentation hosted via GitHub Pages.

## What this is

A simple repo for hosting HTML documents that can be shared via direct links. Each `.html` file pushed to `main` is automatically published.

## How to add a new document

1. Create an `.html` file (e.g. `my-report.html`)
2. Push it to the `main` branch
3. Share the link: `https://cthrnovi.github.io/ayana-docs/my-report.html`

Update `index.html` to add a link to the new document in the doc list.

## Base URL

```
https://cthrnovi.github.io/ayana-docs/
```

Any file at the repo root is accessible at `https://cthrnovi.github.io/ayana-docs/<filename>`.
